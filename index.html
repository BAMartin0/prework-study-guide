HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Elements that are nested inside other elements are child elements.
  • The body of child elements are known as the parent element
  • Semantic HTML element is where the name of the element describes its contents and the role it plays on a webpage.
  • Elements and tags are used interchangably but element is from opening to closing and tag is inside angle brackets.
